# Premier Schools Exhibition Landing Page

A modern, responsive landing page built using HTML5, CSS3, and Vanilla JavaScript based on the provided Figma design.

## 🚀 Features

- Responsive Design
- Modern UI
- Hero Image Slider
- Registration Form
- School Categories
- Participating Schools Section
- Appointment Section
- Benefits Section
- Smooth Animations
- Mobile Friendly
- Cross Browser Compatible

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
